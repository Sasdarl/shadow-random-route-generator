# shadow-random-route-generator
Route selector and tracker for Shadow the Hedgehog's Library Mode.<br />
My first Godot project, and the most niche software I have ever designed.

Controls:<br /><ul>
<li>ENTER/Middle Click/START: Select route</li>
<li>SHIFT/Left Face Button: Mark route complete</li>
<li>BACKSLASH/Top Face Button: Find uncleared route (if possible)</li>
<li>M/SELECT: Toggle BGM and sound effects</li></ul>
